# Sistema Checador Multi-Sede Showcase

Repositorio publico de portafolio para mostrar el alcance del proyecto sin publicar el codigo fuente operativo completo.

## Resumen

Plataforma de asistencia multi-sede con backend en Flask, integracion con WhatsApp Web y utilidades de escritorio para la operacion diaria. El sistema concentra altas de personal, control por sede y departamento, evidencias de registro y exportacion de reportes administrativos.

## Arquitectura mostrada

- backend web en Flask
- bot de WhatsApp Web para flujos operativos
- utilidades de escritorio para arranque y empaquetado
- base de datos MySQL

## Funcionalidades clave

- alta de lugares de trabajo y departamentos
- registro de trabajadores por sede
- identificador unico global
- configuracion de horarios
- vinculacion y monitoreo de dispositivos
- evidencia por registro con texto, foto, mapa y nota de voz
- reportes en PDF, Word y Excel

## Stack

- Python
- Flask
- Node.js
- WhatsApp Web
- MySQL
- ReportLab
- openpyxl
- python-docx

## Valor profesional

Este proyecto muestra capacidad para integrar canales de mensajeria, backend administrativo, generacion documental y control operacional en una sola solucion.

## Alcance publico

El repositorio publico se limita a documentar el proyecto para portafolio. El codigo fuente completo, configuraciones internas, credenciales, sesiones, despliegues y datos operativos no se publican aqui.
